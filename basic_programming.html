"=" IS NOT "equal to", "=" means give some thing a value.

        eg.                 a = 1

                              b = 2
                           
                              sum = a + b                          
                

 "sum" is been given a value of "a + b", so sum is 3.


In a digital world there are only two things

                                        1      and       0

1 is always the opposite with 0

if 0 means no, then 1 means yes
if 0 means negative, then 1 means positive


deal with chips, they are too stupid to understand Arabic number 1 and 0
then what do they understand?
electric pulse!
when there is a signal(electric pulse) then it is 1
when there isn't a signal(electric pulse) then it is 0

if there is a signal present, we call it "HIGH"(means yes, there is a signal!)
if there isn't a signal present, we call it "LOW"(means no, there isn't a signal!)
